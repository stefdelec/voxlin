Voxlin

A decentralized protocol for secure wallet address management

Overview

Voxlin is a cutting-edge protocol that enables users to securely manage multiple wallet addresses while maintaining security and control. By leveraging cryptographic proofs, Voxlin ensures that users can prove the ownership of their wallet addresses with only one wallet.

Key Features

  • Domain Name Linking: Voxlin allows users to link multiple wallet addresses to a domain name. These links can be registered for free.
  • Flexible Domain Management: Domains are acquired through payment in VLN for a specific period of time. After the time period expires, users can retain their linked addresses and assign them to a new domain name. Domains can be sold without affecting the linked addresses.
  • Multiple Assignments: A single set of linked addresses can be assigned to multiple domain names, providing flexibility in managing your digital identity across various platforms.
  • Soulbound Tokens: Each domain name comes with a Soulbound token, which serves as the user's profile. This profile can include links to a website, avatar images, and more, providing a highly customizable user experience.
  • Community driven: Voxlin is by design a decentralized governance and user centric

Main differences with ENS

  • Multiple Wallet Association: Voxlin allows users to link multiple wallet addresses to a single domain name. This is a fundamental difference from ENS, which primarily maps one Ethereum address to a domain name. This feature allows Voxlin users to manage multiple addresses under a single domain name, which can be advantageous for users who manage multiple addresses for different purposes.
  • Domain Management: With Voxlin, users can retain their linked addresses even after the expiration of the domain name. They can assign these linked addresses to a new domain name. This is a feature not currently provided by ENS. This gives Voxlin users greater flexibility and control over their domain names and linked addresses.
